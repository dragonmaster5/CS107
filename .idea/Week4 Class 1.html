Meta Elements

Meta Elements: are tags that define the meta data of a website, that search engines look at to organize it in the search engine. Search engines use automated programs to scan the web for meta data and catalog that data for later use in search. These programs are called Spiders or Webbots

Mailto Links:

Mail to links are an attribute of an anchor element. eg. Contact Us
